<h1>Applied Cryptography Laboratory</h1>

<h2>Cryptographic Algorithms, Protocols & Secure Communication Labs</h2>

<p>
  This repository contains laboratory projects completed as part of the
  <strong>M1-ISC-CSSE-CYBERUS Applied Cryptography</strong> course at the
  University of Southern Brittany (UBS).
</p>

<p>
  The labs focus on implementing and analyzing modern cryptographic algorithms,
  stream ciphers, AES encryption, RSA, ECC, digital signatures, cryptographic
  protocols, and secure communication systems using Python.
</p>

<hr>

<h2>📘 Course Overview</h2>

<p>
  These laboratory exercises explore both theoretical and practical aspects of
  cryptography through hands-on implementations and security analysis.
</p>

<p>
  The projects include:
</p>

<ul>
  <li>Cryptographic hash functions</li>
  <li>Stream ciphers using LFSRs</li>
  <li>AES encryption and modes of operation</li>
  <li>RSA key generation and encryption</li>
  <li>Elliptic Curve Cryptography (ECC)</li>
  <li>Digital signatures</li>
  <li>Diffie–Hellman key exchange</li>
  <li>Hybrid cryptographic protocols</li>
</ul>

<hr>

<h2>🔬 Labs Included</h2>

<h3>Lab 1 – Hash Functions & Stream Ciphers</h3>

<ul>
  <li>Cryptographic hash functions</li>
  <li>Preimage and collision analysis</li>
  <li>Avalanche effect demonstrations</li>
  <li>LFSR (Linear Feedback Shift Register) implementation</li>
  <li>Primitive vs non-primitive polynomials</li>
  <li>Custom stream cipher using XOR and LFSRs</li>
</ul>

<h3>Lab 2 – AES Algorithm & Modes of Operation</h3>

<ul>
  <li>AES-128 implementation</li>
  <li>Key expansion algorithm</li>
  <li>SubBytes, ShiftRows, MixColumns</li>
  <li>AES encryption and decryption</li>
  <li>ECB, CBC, CTR, CFB, OFB, and GCM modes</li>
  <li>File encryption experiments</li>
</ul>

<h3>Lab 3 Part 1 – Asymmetric Cryptography</h3>

<ul>
  <li>RSA key generation</li>
  <li>Rabin-Miller primality testing</li>
  <li>RSA encryption and decryption</li>
  <li>ECC digital signatures</li>
  <li>SHA-256 hashing</li>
  <li>File-based encryption/decryption</li>
</ul>

<h3>Lab 3 Part 2 – Cryptographic Protocols</h3>

<ul>
  <li>RSA-OAEP key transport protocol</li>
  <li>AES-GCM authenticated encryption</li>
  <li>RSA-PSS digital signatures</li>
  <li>Diffie–Hellman key exchange</li>
  <li>HKDF key derivation</li>
  <li>Manual AES integration into protocols</li>
  <li>Encrypt-then-MAC construction</li>
</ul>
